# AWS Cloud Computing Practice ☁️

This repository contains my hands-on practice and learning of **Amazon Web Services (AWS)**.
The goal of this repository is to understand AWS cloud services through practical implementation and document each service with clear explanations and screenshots.

In this repository, I practice creating and configuring various AWS services such as **EC2, S3, IAM, VPC, and more**, while explaining the steps involved in setting them up.

Each service includes:

* Step-by-step explanation of the setup
* Screenshots of the AWS console
* Key concepts learned during the practice

This repository serves as my **AWS learning journal and portfolio**, helping me strengthen my cloud computing knowledge and showcase my practical skills.
